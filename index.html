QUDA: A library for QCD on GPUs

QUDA is a library for performing calculations in lattice QCD on graphics processing units (GPUs) using NVIDIA's "C for CUDA" API. The current release includes optimized solvers for the following fermion actions:

  • Wilson
  • Clover-improved Wilson
  • Twisted mass
  • Improved staggered (asqtad or HISQ)
  • Domain wall
Mixed-precision implementations of both CG and BiCGstab are provided, with support for double, single, and half (16-bit fixed-point) precision. The staggered implementation additionally includes support for asqtad link fattening, force terms for the asqtad fermion action and one-loop improved Symanzik gauge action, and a multi-shift CG solver.
